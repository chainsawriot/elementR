How to install elementR package

###### For Windows and Linux platforms ######

1. Check the version of your R software (≥ 3.2.3 required)

2. Install and source "devtools" package from CRAN by running on R console: 

install.packages("devtools", dependencies = T)
library(devtools)

3. Install and source "elementR" package from GitHub by running on R console:

install_github("charlottesirot/elementR", ref = "master", force = T, dependencies = T)
library(elementR)

Note:
-> elementR runs on any web browser. However, as the graphic of the user interface has been developed based on Firefox, authors recommend to run elementR through this last web browser for an optimal design.

###### For Mac OS platform ######

1. Check the version of your R software (≥ 3.2.3 required)

2. Check that Xcode is installed and runs properly on your computer.

3. Install and source "devtools" package from CRAN by running on R console: 

install.packages("devtools", dependencies = T)
library(devtools)

4. Install and source "elementR" package from GitHub by running on R console:

install_github("charlottesirot/elementR", ref = "master", force = T, dependencies = T)
library(elementR)

Note:
-> If the installation of the elementR package gets stuck during the installation of the package "tcltk", this means that your Xcode is not running properly
-> elementR runs on any web browser. However, as the graphic of the user interface has been developed based on Firefox, authors recommend to run elementR through this last web browser for an optimal design.
